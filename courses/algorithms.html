Course outline:

  1. Intro and setup.
    1. In class work will be done in Java.
    2. Homework:
      1. You are welcome to use other languages for solutions.
      2. I can very readily support Java, Matlab, and C (C++). Other languages I can try and help, but no guarantee there.
      3. Homework skeleton will be given in Java. But, again: It is just a skeleton. You can move to another language.
    3. Topics: Setup the environment: JDK, IDE, Editor, Github.
      1. This one does it all: Installation_NetBeans_JDK_Eclipse.pdf
      2. Just if you need: JDK: Oracle JavaSE IDE - NetBeans: https://netbeans.org/ Screen-shot
      3. Editors:
        • Atom [https://atom.io/]
        • Sublime [https://www.sublimetext.com/]
      4. Netbeans and Git (Clone) ; https://github.com/
    4. In class activity: Syllabus. Installation.


  2. Taxicab numbers: Exercise. Ramanujan and Hardy. [ github]


  3. Numerical alorithms : Introduction to algorithms
    1. Gauss-Jordan : Solution of system of equations.
      1. Gauss elimination: 2x2 ; General case.
      2. Gauss-Jordan: General case.
    2. Newton-Raphson : Finding the zero of a function.
    3. Time/memory : Closest sum exercise.


  4. Dynamic programming : String matching
    1. Levenshtein distance.
    2. Smith-Waterman algorithm.
    3. Exercises with dynamic programming:
      1. Exact change
      2. Partitioning a set of integers into two groups of equal sum


  5. Graph algorithms
    1. Graphs, and graph representation.
    2. Dijkstra : shortest distance.
    3. Minimu spanning trees
    4. Depth first/ breadth first traversal
    5. 8 queens problem
